# Mit første projekt: Portfolio i Hugo

I dette projekt har jeg bygget mit eget portfolio site med Hugo.

##  Formål
Formålet var at lære:
- Hvordan man arbejder med static site generators
- Struktur i projekter
- Deployment

##  Teknologier
- Hugo
- HTML / CSS
- Git & GitHub

##  Hvad jeg lærte
- Hvordan Hugo bruger content og templates
- Hvordan man organiserer et projekt
- Hvordan man deployer et site

##  Udfordringer
Jeg havde især problemer med:
- At forstå Hugo’s struktur
- Styling og layout

Jeg løste det ved at:
- Læse dokumentation
- Eksperimentere med temaer

##  Resultat
Jeg har nu et fungerende portfolio site, som jeg kan bruge til at vise mine projekter.
